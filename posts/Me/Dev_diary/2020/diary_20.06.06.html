Learn

This is my learning things on June 06, 2020.

  • [Github] Why can't I show my contribution on my github page?
    • ANS) Because my local email didn't register. If I want to apply my contribution on github page, local email should be same as github email.
      Then, how to check local email is same as github email? Please enter this command on shell. (In my case, I'm user of WINDOW10, enter commands on git bash.)

      $ git config --global user.email
      $ git config --local user.email

      If it is not same, please register your email.

      $ git config --local user.email username@domain.com
